1 Install the requirements (once)
Open a terminal and run:
pip install seleniumbase seleniumbase install chromedriver
2 Download the setup script
Save this file somewhere easy to find, like your Desktop or Downloads folder.
↓  Download garmin_setup.py
3 Run the script
In your terminal, navigate to where you saved the script and run:
python garmin_setup.py --server-url https://your-app.up.railway.app
Chrome will open and log you in. Enter your MFA code if prompted.
4 Paste your MCP URL into Claude
When the script finishes it will print a URL like:
https://your-app.up.railway.app/garmin/?token=abc123…
Paste that URL into Claude Desktop, Claude.ai, or any MCP-compatible client as your Garmin connector URL.
